match `$ARGUMENTS`.

Behavior:

- Treat `$ARGUMENTS` as fuzzy search terms for titles (e.g. "Q1 plan", "Claude marketplace spec").
- Search both:
  - Individual pages
  - Databases
- Return a short list of the best matches with:
  - Title
  - Type (page or database)
  - Location / parent (if available)
